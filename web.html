A propos le terme Web

web:

Un site web, site Web1,2, est un ensemble de pages web et de ressources reliées par des hyperliens, défini et accessible par une adresse web. .

URL:

Une URL, est une chaîne de caractères uniforme qui permet d'identifier une ressource Web.

HTTP:

L’Hypertext Transfer Protocol est un protocole de communication client-serveur développé pour le World Wide Web.

Hyberlin:

Un hyperlien ou lien hypertexte, est une référence dans un système hypertexte permettant de passer automatiquement d'un document consulté à un autre document
